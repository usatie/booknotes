# 2.2 Declare variables
- Use `const` and `let` instead of `var`
- Problems of `var`
1. `var` can be redeclared
2. Hoisting is allowed for `var`
3. Scope of `var` is function level

- `const` for the first choice
- If you need to reassign, use `let`
- Never use `var`
